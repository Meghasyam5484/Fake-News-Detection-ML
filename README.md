📊 Dataset

This project uses the Kaggle "Fake and Real News" dataset to train and evaluate a machine learning model for fake news detection.

Dataset Source:
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

Files used:
• Fake.csv – Contains fake news articles
• True.csv – Contains real news articles

Note: The dataset files are not included in this repository due to GitHub storage limitations. 
You can download the dataset directly from the Kaggle link above.
